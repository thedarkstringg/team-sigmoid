## Project Overview
Implementation from scratch of three ML algorithms вЂ” Decision Tree (CART), AdaBoost (SAMME), and Random Forest вЂ” plus an unsupervised analysis pipeline (PCA, K-Means, DBSCAN). Controlled empirical study comparing boosting vs. bagging across four real-world datasets.

## Setup

```bash
# 1. Clone the repository
git clone https://github.com/thedarkstringg/ml-final.git
cd ml-final

# 2. Create and activate a virtual environment
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download datasets (optional вЂ” sklearn datasets are fetched automatically)
bash download_data.sh
```

## Reproducing All Experiments

```bash
python src/experiments/run_all.py
```

This runs all 7 experiments and saves figures to `figures/` and results to `results/`.

Expected runtime: ~30вЂ“50 minutes on full datasets (experiments 3вЂ“5 dominate).

## Modules

### Module 1 вЂ” Decision Tree (CART)
`src/trees/decision_tree.py` вЂ” binary CART classifier with Gini/entropy, weighted samples, `max_features` subsampling, OOB-compatible interface.

### Module 2 вЂ” AdaBoost (SAMME)
`src/boosting/adaboost.py` вЂ” discrete SAMME boosting with decision stumps. Supports `staged_predict`, `estimator_weights`, `estimator_errors`.

### Module 3 вЂ” Random Forest
`src/bagging/random_forest.py` вЂ” bootstrap aggregation over decision trees, `oob_score`, parallel training via `multiprocessing.Pool`.

### Module 4 вЂ” Unsupervised Pipeline
- `src/unsupervised/pca.py` вЂ” SVD-based PCA, scree plots
- `src/unsupervised/kmeans.py` вЂ” Lloyd's algorithm with k-means++ init, elbow method
- `src/unsupervised/dbscan.py` вЂ” density-based clustering, k-distance plot
