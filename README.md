## Project Overview
Implementation from scratch of three ML algorithms — Decision Tree (CART), AdaBoost (SAMME), and Random Forest — plus an unsupervised analysis pipeline (PCA, K-Means, DBSCAN). Controlled empirical study comparing boosting vs. bagging across four real-world datasets.

## Setup

```bash
# 1. Clone the repository
git clone https://github.com/thedarkstringg/team-sigmoid.git
cd team-sigmoid

# 2. Create and activate a virtual environment
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download datasets (optional — sklearn datasets are fetched automatically)
bash download_data.sh
```

## Reproducing All Experiments

```bash
python src/experiments/run_all.py
```

This runs all 7 experiments and saves figures to `figures/` and results to `results/`.

Expected runtime: ~30–50 minutes on full datasets (experiments 3–5 dominate).

## Running Tests

```bash
pytest tests/ -v --cov=src --cov-report=term-missing
```

## Modules

### Module 1 — Decision Tree (CART)
`src/trees/decision_tree.py` — binary CART classifier with Gini/entropy, weighted samples, `max_features` subsampling, OOB-compatible interface.

### Module 2 — AdaBoost (SAMME)
`src/boosting/adaboost.py` — discrete SAMME boosting with decision stumps. Supports `staged_predict`, `estimator_weights`, `estimator_errors`.

### Module 3 — Random Forest
`src/bagging/random_forest.py` — bootstrap aggregation over decision trees, `oob_score`, parallel training via `multiprocessing.Pool`.

### Module 4 — Unsupervised Pipeline
- `src/unsupervised/pca.py` — SVD-based PCA, scree plots
- `src/unsupervised/kmeans.py` — Lloyd's algorithm with k-means++ init, elbow method
- `src/unsupervised/dbscan.py` — density-based clustering, k-distance plot
